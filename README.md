# Cyber-Bullying Comments Classifier
7th BOAZ BIG DATA CONFERENCE에서 발표한 프로젝트

### 데이터
데이터와 크롤링 코드는 저작권 문제가 있어 커밋하지 않았습니다.

* Article : 네이버 뉴스 기사(범주, 제목, 발행일시, 연령비, 성비, 감정표현, 본문)
* Comment : 네이버 뉴스 댓글(댓글, 작성일시, 좋아요, 싫어요)
